[https://pick-a-nimal.web.app/](https://pick-a-nimal.web.app/)

```sh
npm install && npm run dev
```

## 스택
- astro
- vue.js 3
- supabase (postgresql)

## 소개

치매예방 게임으로 유명한, 카드 매칭 게임의 웹 버전.
